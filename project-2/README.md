# 🔐 Random Password Generator (Beginner - CLI)

## 📌 Project Overview

This project is a simple **command-line password generator** built using Python.
It allows users to create secure and random passwords based on their preferences such as length and character types.

---

## 🎯 Features

* Generate random passwords
* User-defined password length
* Option to include:

  * Letters (A–Z, a–z)
  * Numbers (0–9)
  * Symbols (!, @, #, etc.)
* Input validation for better user experience

---

## 🧠 Concepts Used

* Randomization (`random` module)
* String handling (`string` module)
* Conditional statements
* User input handling

---

## 💻 Technologies Used

* Python 3

---

## 🚀 How to Run

1. Install Python (if not already installed)
2. Clone this repository or download the code
3. Open terminal in the project folder
4. Run the script:

```bash
python main.py
```
