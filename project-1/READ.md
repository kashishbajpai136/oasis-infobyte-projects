# 🧮 Advanced BMI Calculator (GUI) 🚀

## 📌 Project Overview
This project is a graphical BMI (Body Mass Index) calculator built using Python. It allows users to input their weight and height, calculate BMI, and visualize their health status.

## 🎯 Features
- User-friendly GUI (Tkinter/PyQt)
- BMI calculation with accurate formula
- Health category classification
- Input validation with error handling
- Data storage for multiple users
- Historical BMI tracking
- Graphical visualization of BMI trends

## 🛠 Technologies Used
- Python
- Tkinter / PyQt
- Matplotlib (for graphs)
- File handling / SQLite (for data storage)

## ⚙️ How It Works
1. User enters weight and height
2. Application validates input
3. BMI is calculated using formula:
   BMI = weight (kg) / height² (m²)
4. Result is displayed with category:
   - Underweight
   - Normal
   - Overweight
   - Obese
5. Data is saved for future tracking
6. Graphs show BMI trends over time

## 📊 BMI Categories
- Underweight: < 18.5  
- Normal: 18.5 – 24.9  
- Overweight: 25 – 29.9  
- Obese: ≥ 30
